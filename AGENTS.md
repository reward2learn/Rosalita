# Rosalita — AGENTS.md

**Not a software project.** This repo stores operational files for the Rosalita Cantina restaurant.

## What's here

| Path | Content |
|------|---------|
| `menu.txt` | Text menu — food items, descriptions, prices (in K = IDR thousands) |
| `Menu/food_menu.png` | Food menu graphic |
| `Menu/drink_menu.png` | Drink menu graphic |
| `Rosallita Cashflow May 24th 2026.xlsx` | Cashflow workbook |

## Constraints

- **No code, no build, no tests.** There are no package managers, scripts, linters, typecheckers, or CI pipelines.
- **No git repo.** This directory is not version-controlled.
- **Prices are in IDR thousands** (e.g., `98 K` = 98,000 IDR).
- Never modify images or spreadsheet files — they are binary assets. Edit `menu.txt` only when the printed menu changes.
